Projects

Project 1: DDoS Attack Defender (Cybersecurity)

This project is like a security guard for a website. Its job is to protect a website from being overwhelmed by a huge number of fake visitors (a DDoS attack). It uses a smart system to tell the difference between real users and fake ones. When it spots the fake visitors trying to crash the site, it blocks them so the website stays online and works for everyone.

Project 2: Smart Network Traffic Cop (Networking)

This project is like a traffic controller for internet data. On a computer network, data needs to travel from one point to another. My project figures out the fastest and most efficient roads for this data to take. If one road gets busy, it automatically redirects the data to a clearer path. This helps the entire network run smoothly and without slowdowns.
